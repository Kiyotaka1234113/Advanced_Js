# Lab 8.1: Flyweight Pattern

**Студент:** [Kabylbek Zhantore]  
**Дата:** 08.03.2026

## Описание
Паттерн Flyweight используется для минимизации потребления памяти при рендеринге 10 000 объектов (деревьев). 
- **Intrinsic State** (внутреннее состояние) разделяется между множеством объектов через `TreeFactory` (цвет, тип, форма).
- **Extrinsic State** (внешнее) передается клиентом (координаты X, Y).

## Результаты бенчмарка
Количество уникальных объектов в памяти сокращено практически на 100% за счет переиспользования 4 базовых типов деревьев.
