3. CommonJS Note

CommonJS uses require() and module.exports, which are native to Node.js. To run the CommonJS module in the browser, a bundler like Webpack or Browserify is required. Test it via terminal: node -e "const lib = require('./modules/commonjs/MyLib.js'); console.log(lib.version);"
