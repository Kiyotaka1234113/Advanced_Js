* Формы входа (логина)
* Подписки на рассылки 
* Любая клиентская валидация перед отправкой данный на сервер 
### Обзор решения 
- Функция проверяет соответсвует ли переданная строка стандартному формата email перед дальнейшей обработкой 
### Как функция решает проблему 
# 1.Принимает строку в качестве входных данных
# 2.Использует регулярное выражение для проверки формата email
# 3.Возвращает логическое значение (true/falce)
### Ключевые компоненты 
* Входная строка (email)
* Регулярное выражение для валидации
* Логическое возвращаемое значение
